# bs-dom

bs-dom is a DOM library for [BuckleScript](https://github.com/bloomberg/bucklescript).  
This Library includes modified [Js_of_ocaml](https://github.com/ocsigen/js_of_ocaml).  
Dom types are the same as js_of_ocaml.   

## requirements
  BuckleScript

## Installation
npm install bs-dom
